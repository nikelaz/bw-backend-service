# CategoryBudgets POST /

## Request Body

| Key | Type | Required? |
|-|-|-|
| categoryBudget | CategoryBudget | `true` |
| categoryBudget.amount | number |
| categoryBudget.transactions | Array&lt;Transaction&gt; |
| categoryBudget.currentAmount | number |
| categoryBudget.category | Category |
| categoryBudget.category.id | number |
| categoryBudget.category.type | number |
| categoryBudget.category.title | string |

**Note:** if creating a CategoryBudget in an existing category, pass the id property, otherwise add the type and title properties and the new category will be created

## Response Body

| Key | Type |
|-|-|
| categoryBudget | CategoryBudget |
| categoryBudget.id | number |
| categoryBudget.amount | number |
| categoryBudget.transactions | Array&lt;Transaction&gt; |
| categoryBudget.currentAmount | number |
| categoryBudget.category | Category |
| categoryBudget.category.id | number |
| categoryBudget.category.type | number |
| categoryBudget.category.title | string |
| categoryBudget.category.accAmount | number |
